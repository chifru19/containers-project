# Containers Project
This project demonstrates containerization, deployment, and orchestration using Docker and Kubernetes.

## Key Features
- **Containerization:** Built and deployed a web application using Docker.
- **Orchestration:** Managed deployment via Kubernetes, including:
    - **Deployment:** For scalable application instances.
    - **DaemonSet:** For ensuring the application runs on all cluster nodes.
    - **HPA (Horizontal Pod Autoscaler):** Configured to automatically scale pods based on CPU utilization.
- **Configuration:** Managed application secrets and configuration maps for secure, flexible environments.

## Evidence
- Deployment logs and image verification are tracked in `deployment_evidence.txt`.
- Dependencies are listed in `requirements.txt`.
